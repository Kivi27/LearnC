<h2>How to compile?</h2>
<h3>For C file</h3>
<h3>template</h3>
gcc -Wall -g file.c -o file -l some_library<br>
<b>example:</b> <br>
gcc -Wall -g triangle.c -o triangle -lm <br>
m = math library for C <br>
<h3>For C++ file </h3>
<br>
g++ file.cpp -o file
